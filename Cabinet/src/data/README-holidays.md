# holidays.json

未来 **70 年**（2026–2095）节日 **日期表**（北京时间自然日 `YYYY-MM-DD`）。

不含国务院放假调休安排，仅节日当日（春节为农历正月初一至初八）。

## 规则

| 节日 | 规则 |
|------|------|
| 元旦、情人节、劳动节、儿童节、国庆节、圣诞节 | 公历固定月日 |
| 元宵节、端午、七夕、中秋、重阳 | 农历固定月日 |
| 春节 | 农历正月初一至初八 |
| 除夕 | 春节正月初一公历日减 1 天 |
| 清明 | 每年公历 4 月 4 日至 6 日（三天） |

## 重新生成

```bash
pip install lunar-python
npm run generate:holidays
```

生成节日与节气互不影响，可分别执行。
